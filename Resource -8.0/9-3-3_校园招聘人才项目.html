Gensci RUN 金锐实习夏令营

金锐实习夏令营--Gensci RUN诚邀海内外顶尖学府的优秀本科生、博士及MBA在读生加入，我们将为大家提供在金赛总部进行为期8周（每年7月-8月）的暑期实习平台，让更多优秀的学子有机会体验金赛的核心业务、了解金赛的价值主张，同时在实习期内安排丰富的业余生活及分享，让大家在体验业务了解工作之余，进一步的将生活、兴趣与工作完美的结合，在一个真实的商业情境下学习实践，也为金赛集团的未来发展储备一批精锐之师！接下来，金赛回味优秀的你提供金赛集团全职人才项目（TET/IMT）的绿色通道。如果你足够优秀，你将优于其他人提前拿到金赛的全职工作offer！

  • 强烈的责任意识、优秀的沟通能力和团队合作精神
